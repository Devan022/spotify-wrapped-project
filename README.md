# 🎵 Spotify Wrapped – AI Music Insights Platform

## 📌 Overview
A web-based music analytics platform that analyzes user listening data using the Spotify Web API and generates personalized insights similar to Spotify Wrapped.

## 🚀 Features
- Fetches user listening history
- Processes listening patterns
- Displays top artists and tracks
- Music mood analysis

## 🛠 Tech Stack
- Python
- Jupyter Notebook
- Spotify Web API

## ⚙️ How to Run
1. Clone the repository
2. Install required libraries
3. Run the notebook file

## 🎯 Learning Outcome
- API Integration
- Data Analysis
- Visualization
